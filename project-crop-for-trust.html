Crop for Trust: Reliability-Aware Image Classification

딥러닝 모델이 핵심 객체가 아닌 배경의 가상 상관관계(Spurious Correlation)에 의존하는 'Shortcut Learning' 문제를 해결하고자 합니다. 이를 위해 Gen AI와 YOLO를 결합하여 데이터 의존성을 원천적으로 개선하는 강건한 객체 분리 및 정제 프레임워크를 개발했습니다.

Problem Definition

현실의 딥러닝 모델들은 종종 학습 데이터셋에 내재된 '지배적인 배경 노이즈'를 타겟 레이블과 잘못 연관 지어 학습하는 오류를 범합니다. 예를 들어, 특정 클래스의 이미지가 특정 배경과 함께 빈번히 등장할 경우, 모델은 객체의 본질적인 특성이 아닌 배경 텍스처를 판단 근거로 삼게 되며 이는 모델의 본질적인 신뢰성을 심각하게 훼손합니다.

  • Out-of-Distribution 환경에서의 극심한 일반화 성능 저하 발생.
  • 의사결정 과정의 불투명성으로 인한 XAI 관점에서의 신뢰성 및 해석력 결여.
  • 도메인 변이에 취약하여 실제 산업 현장 적용 시 이식성 한계 노출.

Methodology & Architecture

본 연구는 모델 아키텍처의 변경 없이 데이터 자체의 품질을 향상시키는 Data-Centric AI 패러다임을 채택하여, 배경 노이즈를 원천적으로 차단하는 파이프라인을 설계했습니다.

1. GAC (Generative Annotation Cleaning)

Gemini 2.5 Flash 모델의 API를 활용하여, Zero-Shot 방식으로 데이터셋 내에 포함된 시각적 편향 (Visual Marking)을 정밀하게 제거합니다. 이를 통해 모델이 객체 외부의 통계적 특성에 의존하는 것을 방지합니다.

2. ROI Focusing (YOLO-Crop)

YOLO v8 모델을 활용하여 이미지 내 객체의 위치를 탐지하고, 해당 영역을 제외한 배경을 제거합니다. 이를 통해 모델이 배경으로 Attention이 분산되는 현상을 물리적으로 차단하고 객체 고유의 특징에만 집중하도록 강제합니다.

Technical Keywords:

Gen AI (Gemini API) YOLOv2 ~ YOLOv8 Grad-CAM Data-Centric AI Shortcut Learning

Results & Impact

모델의 의사결정 과정을 시각화하는 Grad-CAM 분석 결과, 기존 베이스라인 모델은 배경의 무의미한 텍스처에 강한 Attention을 기울이는 한계를 보였습니다. 반면, 'Crop for Trust' 파이프라인이 적용된 모델은 추론의 근거가 핵심 객체의 형태와 특징으로 정확히 이동하였음을 입증하여, 산업 현장에서 요구되는 높은 해석 가능성과 신뢰성을 확보했습니다.
