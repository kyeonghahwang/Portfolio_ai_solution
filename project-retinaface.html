01
Problem Definition

경량화 시 성능이 급감하는 RetinaFace의 한계

RetinaFace는 고성능(ResNet-152 백본)과 경량화(MobileNet-v1 백본) 사이의 극심한 성능 격차 문제가 있었습니다. 우리는 MobileNet-v1로도 고성능을 낼 수 있는 구조적 개선을 목표로 삼았습니다.

문제 1: 성능 격차

백본을 ResNet-152에서 MobileNet-v1으로 교체 시 AP가 약 13 포인트 급감합니다. 경량화와 고성능을 동시에 달성할 수 없습니다.

ResNet-152: 고성능 MobileNet-v1: −13 AP

문제 2: 병목 구조 (Detection Head)

RetinaNet의 경량화 후속 논문 분석 결과, Detection Head 초기 부분에 FLOPs가 집중됨을 확인했습니다. 중간 레이어의 정보 전달 효율을 높이면 성능을 끌어올릴 수 있다고 판단했습니다.

ResNet-152 vs. MobileNet-v1 성능 비교 (WiderFace Benchmark)

02
Proposed Architecture

4개의 커스텀 모듈로 재설계한 탐지 파이프라인

기존 구조

MobileNet v1 → FPN → SSH → Detection Head

기존 RetinaFace 아키텍처 전체 구조도

개선 구조 (Ours)

MobileNet v1 → ECA-CBAM → WFPN → ECA-CBAM → SCM → ECA-CBAM → Detection Head

* 모든 Convolution을 Deformable Convolution으로 대체

제안하는 개선 아키텍처 전체 구조도

핵심 개선 모듈 상세

Module 01

FPN (Feature Pyramid Network) → WFPN (Weighted Feature Pyramid Network)

기존 FPN은 고수준(high-level) 특징을 저수준(low-level) 특징으로 단순 합산(sum)하여 흘려줍니다. 이 과정에서 두 특징의 중요도가 동일하게 취급되어 정보 손실이 발생합니다.

WFPN은 두 특징 사이에 학습 가능한 가중치(Learnable Parameter)를 도입하여, 상황에 따라 고수준/저수준 정보의 기여도를 동적으로 조절합니다. 이를 통해 정보 손실을 줄이고 특징 피라미드의 표현력을 향상시킵니다.

// FPN
out = high_feat + low_feat

// WFPN (Ours)
out = w₁ × high_feat + w₂ × low_feat // w₁, w₂ are learnable
Module 02

ECA-CBAM (Efficient Channel Attention CBAM)

중간 레이어 간 정보 흐름을 정제하기 위해 공간·채널 Attention 모듈을 FPN 전·후, SCM 후 총 3곳에 삽입했습니다.

CBAM 채택 이유
공간 Attention + 채널 Attention을 동시 적용하고, Adaptive Kernel Size로 연산 효율이 높습니다.
ECA 커스터마이징
Channel Attention의 MLP Layer를 1D Convolution으로 대체하여 연산 부담을 줄이면서 표현력을 유지합니다.

[Figure 5] ECA-CBAM 모듈 구조: 1D Conv를 활용한 효율적인 Attention 메커니즘

Module 03

SSH → SCM (Shuffle Channel Module)

기존 SSH는 3×3, 5×5, 7×7 Convolution으로 다양한 receptive field를 추출하지만, 채널 수를 줄이는 구조가 없어 연산량이 상대적으로 큽니다.

항목 기존 SSH SCM (Ours)
Receptive Fields 3×3, 5×5, 7×7 3×3, 5×5, 7×7, 9×9
채널 수 그대로 유지 축소 (연산 절감)
결합 방식 Concatenation Channel Shuffle

채널 Shuffle 과정을 통해 더 넓은 receptive field의 정보를 효율적으로 결합합니다.

Module 04

Deformable Convolution

모든 일반 Convolution을 Deformable Convolution으로 대체했습니다. 표준 Conv는 격자형 고정 위치를 샘플링하지만, Deformable Conv는 offset 기반 동적 샘플링으로 얼굴의 변형(포즈, 가림, 소형 얼굴)에 robust하게 대응합니다.
